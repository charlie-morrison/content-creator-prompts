# Content Creator AI Prompts

A collection of AI prompts for content creators — covering blog writing, social media, email marketing, video scripts, and SEO. Works with ChatGPT, Claude, Gemini, or any AI assistant.

Fill in the `[BRACKETS]` with your details and go.

## Blog Writing

### Topic Brainstorm
```
Generate 15 blog post ideas about [NICHE] that would interest
[TARGET AUDIENCE]. Focus on problems they're actively searching
for solutions to. For each idea, include a working title and
the specific pain point it addresses.
```

### Outline Generator
```
Create a detailed blog post outline for: '[TITLE]'.
Include: intro hook (2 options), 5-7 main sections with subpoints,
places to add examples or data, a conclusion with CTA.
Target word count: [NUMBER].
```

### Hook Writer
```
Write 5 different opening hooks for a blog post titled '[TITLE]'.
Use these styles:
(1) startling statistic
(2) contrarian take
(3) personal story format
(4) question that creates curiosity
(5) bold claim with promise
```

### Twitter/X Thread Converter
```
Convert this blog post into a 7-tweet thread. First tweet should
be a hook that stands alone. Each subsequent tweet should deliver
one key insight. Last tweet should link to the full post:
[PASTE POST]
```

## Social Media

### Monthly Content Calendar
```
Create a 30-day social media content calendar for [PLATFORM]
in [NICHE]. Include: post type (carousel, text, video, poll),
topic, best posting time, and which content pillar it serves.
My pillars are: [LIST 3-4 PILLARS].
```

### Thread Starter
```
Write a Twitter/X thread (8-10 tweets) about [TOPIC].
First tweet must hook — use a bold claim, surprising stat,
or 'I spent X hours so you don't have to' format.
Each tweet should be under 280 chars and deliver standalone value.
```

### LinkedIn Post (Story Format)
```
Write a LinkedIn post using the story-lesson format about
[EXPERIENCE/INSIGHT]. Structure: hook line → 2-3 paragraph
story → clear takeaway → question for engagement.
Under 1300 characters. Avoid corporate jargon.
```

### Instagram Carousel Script
```
Create a 7-slide Instagram carousel about [TOPIC].
Slide 1: attention-grabbing title with emoji.
Slides 2-6: one tip/point per slide (short, scannable).
Slide 7: CTA to save/share.
Write both the slide text and the caption.
```

### Engagement Tweet Templates
```
Give me 10 tweet templates designed to drive replies and
engagement for someone in [NICHE]. Types: hot takes,
'unpopular opinion', 'what's your X?', fill-in-the-blank,
'this or that', and prediction tweets.
```

## Email Marketing

### Welcome Sequence
```
Write a 5-email welcome sequence for new subscribers to my
[NICHE] newsletter.
- Email 1: Welcome + best content (immediate)
- Email 2: My story + credibility (Day 2)
- Email 3: Quick win / free resource (Day 4)
- Email 4: Social proof + community (Day 7)
- Email 5: Soft pitch for [PRODUCT/SERVICE] (Day 10)

For each: subject line, preview text, body (under 200 words), CTA.
```

### Newsletter Template
```
Write this week's newsletter issue:
- Topic: [TOPIC]
- Key insight: [MAIN POINT]
- Audience: [WHO THEY ARE]
- Tone: [casual/professional/witty]

Structure: hook opening, 3 actionable points, one personal take,
CTA. Include subject line and preview text. Under 500 words.
```

### Re-engagement Campaign
```
Write a 3-email re-engagement sequence for subscribers who
haven't opened in 60+ days.
- Email 1: 'We miss you' + best recent content
- Email 2: Quick survey (why they stopped reading)
- Email 3: Final 'stay or go' with cleanup warning

Make each feel personal, not automated.
```

## Video & Podcast

### YouTube Script Outline
```
Create a YouTube video script outline for: '[VIDEO TITLE]'.
Target length: [X] minutes.
Include:
- Hook (first 15 seconds — what to say to stop scrolling)
- Intro (why this matters)
- Main content (3-5 sections with key points)
- B-roll/visual suggestions for each section
- CTA (subscribe, comment prompt, related video)
```

### TikTok/Reel Script
```
Write a 60-second script for a [PLATFORM] video about [TOPIC].
- Hook (first 3 seconds): [what to say AND what to show]
- Main content: 3 quick points with visual cues
- CTA: what the viewer should do next
Format: timestamps, spoken text, on-screen text suggestions.
```

## SEO & Copywriting

### Meta Description Generator
```
Write 3 meta descriptions (under 155 characters each) for my
blog post: '[TITLE]'. Each should include the primary keyword
'[KEYWORD]', create curiosity, and include a clear benefit
for clicking.
```

### Landing Page Copy
```
Write landing page copy for [PRODUCT/SERVICE].
- Target audience: [WHO]
- Main problem it solves: [PROBLEM]
- Key benefits: [LIST 3-5]
- Price: [AMOUNT]
- Social proof: [TESTIMONIALS/STATS]

Structure: headline, subheadline, 3 benefit blocks, objection
handling, testimonial section, FAQ, and CTA. Above-the-fold
copy should convert scanners.
```

### Headline Analyzer Prompt
```
Rate these headlines for my blog post and suggest improvements:
[LIST 5 HEADLINES]

Score each on: clarity, curiosity, specificity, emotional pull,
and SEO potential. Then write 3 better alternatives.
```
